# Predictive Modeling of Diabetes Risk Factors using CDC-BRFSS Data and Classification-Algorithms
A machine learning project that uses 250k+ CDC BRFSS health records to build a multi-class classifier (76.3% Acc, 0.74 ROC-AUC) for predicting diabetes risk and identifying high-impact lifestyle factors.
